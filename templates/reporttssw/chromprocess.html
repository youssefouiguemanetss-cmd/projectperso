Project Description

Chrome Connect is a Chrome extension designed to automate the Gmail account connection process. The project handles bulk Gmail accounts by creating Chrome profiles, installing the extension on each profile, and launching an automated connection process. The extension manages email entry, password verification, recovery email handling, and tracks connection status for each account. Using AES-GCM encryption for secure credential storage, the system provides comprehensive logging and error handling for various failure scenarios including wrong credentials, captcha challenges, and proxy issues.
